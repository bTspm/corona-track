### Installation
- Requires [Ruby](https://www.ruby-lang.org/en/downloads/) 2.7.0
- Requires [Rails] 5.2.3

Install the dependencies and start the server.

```sh
$ cd corona-track
$ bundle install
$ rails s
```

### Tests
```sh
$ bundle exec rspec spec
```
